# ProductionFXMLProject
This project is a product logging application that can add new product, and log the production of products already stored in the database. 
I developed the GUI the backend and the database. This project was created to demonstrate the skills learned in COP 3003 (object-oriented programming) my junior year at FGCU. 
This was a solo adventure that was created august 26 2020.
